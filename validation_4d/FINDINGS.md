# 4D 时序真预测验证：尝试与诚实结论（2026-06-14）

## 目标
真预测验证：播种循环 N 的位错构型 → PFC 演化 → 比对循环 N+Δ 的实际数据。

## 数据勘察
- group 重建（67,68 / 102,108…）= **立体对单时刻重建**（两视图匹配），非时间帧。
- 真正的 4D 时序在 Interface-B HDF5：210 循环的 ROI 密度 rho_total_avg + 驱动应力 sigma_avg。
- **三个 ROI 文件全部密度单调上升（209 升 0 降，连逐胞 0% 恢复），sigma_vm 0→1200 MPa**
  = 室温无热单调加载（储存主导）。driving 为逐循环汇总，循环内恢复被平均掉。

## 诚实结论：区制错配，严格验证对此数据集不适用
**实验/DAMASK 数据所在区制（无热单调硬化）= 恰恰是 PFC 不能捕获的区制**
（PFC 有效区 = 扩散/恢复/蠕变，已三方确认：森林硬化不可达 + 钉扎诊断 + 此处）。
- 直接预测净 ρ：PFC 会处处低估（无储存），匹配只会是巧合 → 不做（不强凑）。
- 这从第三个方向再次实证确认了区制错配——是精确的科学结论，非验证方法失败。

## 验证 PFC 桥所需的数据（明确给出）
PFC 桥要被真预测验证，需**恢复/扩散主导**的时序位错数据：
1. **高温蠕变**实验（恒载下位错密度随时间，恢复显著）— 最直接锚定 PFC 迁移率/恢复；
2. **应力弛豫/保载**实验（恒应变,应力随时间衰减,位错恢复)；
3. **循环内细时间**位错密度（卸载段恢复),非逐循环汇总。
有了任一种,本桥的播种管道(experiment_bridge,已验证忠实)即可直接做真预测验证。

## 元结论
"真预测验证"诚实地产出了一个精确负结果 + 验证实验的明确规格——这比强凑一个区制外的
匹配有价值。它精确界定了 PFC 多尺度桥的可用边界:高温蠕变/恢复 yes,室温无热硬化 no。
