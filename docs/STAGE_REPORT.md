# 阶段性报告：PFC 模拟拉伸条件下的位错演化机制

> 2026-06-11 · 迭代 1-7 · 全部本地计算（HPC 始终不可达）
> 仓库: github.com/Shirakawa11111/Multiscale-metal-simulation

## 1. 方法

**模型**：标准 Elder–Grant PFC，F = ∫[ψ/2(r+(1+∇²)²)ψ + ψ⁴/4]，保守动力学。
参数 r=-0.25、ψ̄=-0.25（2D 三角相 / 3D BCC 相）。

**数值格式**（经 3 轮失稳排查后定型）：
- rfft 谱方法 + **实场为状态变量、每步完整往返投影**（消除幻影谱分量在 k≈1 放大带的无饱和增长——曾导致固定物理时间 t≈400 的条纹爆炸）
- 稳定化分裂：C=3max ψ² 项隐式处理
- 验证：512² 单次 6000 步健康，~4ms/步（pyFFTW 8 线程）

**加载**：盒子仿射应变（k 重标定），面积/体积守恒单轴拉伸，0.25%/步 + 步间弛豫（弛豫步数 = 有效应变率的倒数）。

**应力**：虚拟仿射变形 σ=dF/dε（与独立能量曲率交叉验证：0.209 vs 0.211）。

**位错检测**：峰提取（亚网格 COM 精化）→ 周期 Delaunay → 配位数 → 5|7 对 = 刃位错核心；位错植入用相位绕动法（净 Burgers=0，偶极子/四极子验证精确）。

**尺度映射**（Interface B 对接）：a₀(PFC) ↔ b(Cu)=2.556 Å → 1 PFC 单位=0.0352 nm；512²盒=14.2 nm，1536²盒=42.5 nm。

|---|------|------|----------|
| M1 | 拉伸驱动滑移 | A3b, D2 | 2% 应变滑移 ~4a₀；D2 轨迹全程记录 |

| M3 | **位错增殖** | D2 旗舰 | **10.50% 应变 4→8**，固定应变续弛豫 2000 步验证持久；阈值 σ/E≈7% |
| M4 | GB 塑性 + 率敏感 | B3/B5 | 多晶模量为单晶 39%；流应力 0.0057/0.0036/0.0026（率 4×↑/基准/4×↓）→ **粘塑性率指数 m=0.28**（对应金属扩散/GB 主导蠕变-超塑性区间）；快应变率下位错保有量更高（148/122/102 @8.5%） |

| M7 | 孔洞 Gibbs-Thomson 溶解 | B4 pore_v1 | 小/浅孔洞（depth 0.2、4a₀）~6% 应变前溶解愈合 |

**3D（BCC）**：结晶/弹性拉伸门禁通过（C1/C2）；128³ 多晶（3290 原子）模量为单晶 75%、应力-应变呈早期塑性弯折（C3）。

**256³ 旗舰（M15，HPC，4.3 小时，32 线程）**：拉伸至 6.2%。
关键负面结论：disordered_frac=0.67、单一渗流网络，**与 128³ 完全相同**——
证明 PFC 淬火多晶的**晶粒尺寸由淬火形核密度决定，与盒子无关**（大盒=更多小晶粒，
非更大晶粒）。所以个体位错线分辨不能靠放大盒子，需大晶粒配置（双晶/长退火）。
拉伸下 disordered_frac 微降 0.674→0.663 = 3D 应变辅助 GB 恢复（呼应 M10）。
图：`results_hpc_3d/ANALYSIS/flagship3d.png`

**晶粒粗化（M15 定量验证）**：128³ 多晶退火至 30000 步（10× 标准淬火），
disordered_frac 仅 0.670→0.628（相对降 6%），能量几乎不变、始终单一渗流网络
——**PFC 纳米晶结构基本冻结，晶粒粗化极慢**（守恒动力学无热噪声去钉扎晶界）。
坐实：分辨个体位错线必须走双晶路线，退火粗化无效。图：`results/grain_coarsening/coarsening.png`

**3D 双晶（M16）**：`init_bicrystal`（非 CSL）缺陷集中到晶界但周期边界有拓扑
必需的体相失配残留（GB/体相 ~1.7）。**`init_bicrystal_csl`（Σ5 扭转，θ=atan(3/4)≈36.87°，
5 胞公度，勾股 3-4-5）实现零体相失配**：验证 80×80×160 弛豫后体相无序原子=0，
缺陷全在两个晶界面。拉伸下晶界作为位错源向干净体相发射个体位错线——
3D 版"位错工厂"（运行中）。

**3D 矩阵（M13，HPC nanolab2023，16 单元 128³：r×率×种子）**：
- 流应力对 (r, 率) **双轴单调**，与 2D 同号（冷/快更强）
- 模量单调随 r：0.057(r=-0.35)→0.022(r=-0.20)，**约为 2D 的 2 倍**（BCC 三维约束更刚）
- 3D 率指数 m_3D ≈ 0.15-0.36（仅 2 率点×2 种子，初步；趋势与 2D 同量级）
- 图：`results_hpc_3d/ANALYSIS/matrix3d.png`

**近旋节线 768² 扫描（M14，本地，进行中）**：r=-0.185（距液体边界 -0.1875 仅 0.0025）
**流应力已≈0**——确认逼近旋节线即进入粘性流动，2D m(r) 曲线在此终结于液体。

**3D 位错线检测（M12，自研无 OVITO 依赖）**：基于 BCC 14 近邻壳的结构无序度
（完美晶 0.031、空洞缘 0.13，绝对阈值 0.06 分离）+ 周期连通分量聚类。
C4 门禁过（完美/空洞/应变不变性三态）。应用于真实 128³ 多晶：disordered_frac=0.67、
GB 网络渗流为单一连通块——直接证实该尺度是**纳米晶（GB 主导）**，个体位错线分辨
需更大盒子（→ HPC 256³ 旗舰，晶粒/a₀ 比提高 2 倍）。检测器 `src/defect_analysis_3d.py`。

### 2.1 增殖级联（M3 完整版，D2 续跑至 21.5%）

ROI 密度匹配盒（42.5 nm、初始 4 核心 = 2.2×10¹⁵ m⁻²）完整屈服转变：
- **屈服峰** σ=0.0148（σ/E≈7%，理论强度量级）@ 11.6% 应变
- **位错雪崩**：增殖 4→8（10.5%）→ 级联至峰值 **316 核心**（~17%，ρ=1.75×10¹⁷ m⁻² = 初始 79 倍）
- **流动**：软化至 σ=0.0073 = 屈服峰的 49%，位错回落稳定 ~190（湮灭-增殖动态平衡）
- 图：`results/d2_roi_matched_1536/analysis_cascade.png`

### 2.3 淬火深度（有效温度）依赖（M8，r 扫描）

512² 四极子拉伸至 16%，仅变 r：

| r | 峰值应力 | 增殖阈值 | 末态 σ / 核心 | 响应类型 |
|---|---------|---------|--------------|---------|
| -0.35（冷） | 0.0255 | 11.1% | 0.0201 / 21 | 高强度，有限增殖 |
| -0.25（参考） | 0.0149 | 15.6% | 0.0086 / 39 | 屈服-软化 |
| -0.15（热） | 0.0006 | 0.25% | ≈0 / 47 | **粘性/超塑性流动**（扩散完全吸收应变） |

冷→热 = 强而脆 → 即时塑性流动的连续转变。
注：r=-0.25 阈值在 512²（15.6%）与 1536² D2（10.5%）不同——四极子在小盒中

### 2.4 循环加载：安定行为（M9）

四极子 512²、±3% 对称三角波、8 周期：
- 周期 0-1 塑性耗散主导（回线面积 5.6-7.1×10⁻⁶），位错往返滑移
- **第 1 周期内全部湮灭**（4→0），其后纯粘性本底耗散（~2.3×10⁻⁶ 缓慢衰减）
- = 循环安定（shakedown）：该幅度下缺陷不棘轮累积而是退火消失
- 与单晶铜 dislotwin 循环组织演化的对照：PFC 扩散时间尺度对应高温行为，
  且无持续位错源；多晶循环（GB 为源/汇）见 M10
图：`results/cyclic_512/analysis_cyclic.png`

### 2.5 多晶循环：恢复主导（M10）

多晶 512²、±1%、8 周期（对照四极子 ±3%）：
- GB 网络维持持续位错群体（不归零），但**逐周期缓慢恢复** 156→113
  （~3-5%/周期，递减），耗散 4.9→1.1×10⁻⁶ 同步衰减，趋向稳定回线
- 无棘轮累积 —— 扩散主导（高温极限）下循环辅助恢复胜过缺陷生成
- ±2% 高幅度对照运行中（检验幅度是否翻转行为）
图：`results/cyclic_poly_512/analysis_cyclic_compare.png`

**ROI 密度匹配（多尺度对接核心）**：DAMASK 热点 ROI ρ≈1.5-2.05×10¹⁵ m⁻² → 密度匹配 PFC 盒仅需 40 nm（1400² 网格），本地可跑。D2 旗舰即按 ROI 单元峰值密度配置（4 核心/42.5 nm = 2.2×10¹⁵），其增殖事件已按 CU 框架 DDD event proxy 格式输出（nucleation_count、threshold_strain）。

### 2.6 全参数矩阵机制图（M11，57 单元，本地 11 核 46 分钟）

**多晶 (r × 应变率 × 3 种子，14% 应变处取值)**：
- 流应力图双轴单调：深淬火更强、快应变率更强（0.0124 @冷快 → 0.0012 @临界慢）
- **率指数随淬火深度单调上升：m = 0.19 (r=-0.35) → 0.24 → 0.27 → 0.45 (r=-0.20)**
  ——从位错滑移主导（低 m）连续过渡到接近经典超塑性值（m≈0.5），
  与真实金属趋近熔点时率敏感性升高的规律一致
- 位错保有量：慢应变率退火更充分（0.69→0.46 随率降）
- ⚠️ r=-0.15 行是**液体**（ψ̄=-0.25 处 r+3ψ̄²>0，液相线性稳定，随机淬火不结晶；
  2.3 节 quad 在该点有晶体是种子晶体的亚稳共存态）——固体图谱止于 r=-0.20
- 种子间标准差远小于趋势量级（σ_sd/σ ~ 3-10%）→ 结论统计稳健

**四极子增殖阈值 vs r（v2 细化：9 个 r × 3 系综）**：
- **临界增殖应力 σ_c(r) 严格单调且近线性**（0.0238→0.0052，斜率 -0.112），
  线性外推在 r≈-0.13 归零（熔化区附近增殖阻力消失）
- onset **应变**的非单调（9.4%↔15.6%，r≈-0.27 处台阶）是 σ_c(r)↑×E(r)↑ 的
  簿记效应——物理控制变量是应力不是应变
- 系综 SD≈0：阈值是确定性的（0.01 扰动在阈值前完全退火）

## 4. 已知局限与待办

1. 应力为加载路径功共轭量（PFC 无量纲单位），物理标定需弹性常数换算（C11-C12 等）——可与 DAMASK ROI 的 σ_vm(t)（峰值 ~1380 MPa）做比例对齐
2. 孔洞几何自动检测阈值不可靠（视觉确认正确）→ 改进 pore_geometry
3. 3D 位错线检测未实现（需 3D Burgers 回路或导出 OVITO/DXA）
4. PFC 扩散动力学时间尺度 ≠ MD 惯性时间尺度：应变率只能做相对比较
5. b4_noise_v2 中 cores=1 的奇数计数 = 偶极子部分检出（配对距离阈值）
6. HPC 不可达贯穿全程（7 轮重试）；大规模扫描（r×率×密度矩阵、3D 256³）待 HPC 可用

## 5. 与现有项目的衔接

- **CU 框架**：本工作 = Layer 4b（PFC 支线）首次实装；Interface B 读取器 + 密度种子映射 + 事件指标格式对齐已完成（D1/D2）
- **单晶铜拉伸模拟**：孔洞拉伸设置呼应其弧形缺口几何；σ(ε) 指标族与其 GROD/滑移分析互补
- **Topology research**：5|7 检测方法同源；模量/峰值强度/韧性指标对齐其 analyze_tensile_scan.py
- **BJK_hBN_Multiscale**：XPFC 谱方法数值风格的金属域移植
