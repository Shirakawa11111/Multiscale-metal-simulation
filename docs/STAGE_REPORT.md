# 阶段性报告：PFC 模拟拉伸条件下的位错演化机制

> 2026-06-11 · 迭代 1-7 · 全部本地计算（HPC 始终不可达）
> 仓库: github.com/Shirakawa11111/Multiscale-metal-simulation

## 1. 方法

**模型**：标准 Elder–Grant PFC，F = ∫[ψ/2(r+(1+∇²)²)ψ + ψ⁴/4]，保守动力学。
参数 r=-0.25、ψ̄=-0.25（2D 三角相 / 3D BCC 相）。

**数值格式**（经 3 轮失稳排查后定型）：
- rfft 谱方法 + **实场为状态变量、每步完整往返投影**（消除幻影谱分量在 k≈1 放大带的无饱和增长——曾导致固定物理时间 t≈400 的条纹爆炸）
- 稳定化分裂：C=3max ψ² 项隐式处理
- 验证：512² 单次 6000 步健康，~4ms/步（pyFFTW 8 线程）

**加载**：盒子仿射应变（k 重标定），面积/体积守恒单轴拉伸，0.25%/步 + 步间弛豫（弛豫步数 = 有效应变率的倒数）。

**应力**：虚拟仿射变形 σ=dF/dε（与独立能量曲率交叉验证：0.209 vs 0.211）。

**位错检测**：峰提取（亚网格 COM 精化）→ 周期 Delaunay → 配位数 → 5|7 对 = 刃位错核心；位错植入用相位绕动法（净 Burgers=0，偶极子/四极子验证精确）。

**尺度映射**（Interface B 对接）：a₀(PFC) ↔ b(Cu)=2.556 Å → 1 PFC 单位=0.0352 nm；512²盒=14.2 nm，1536²盒=42.5 nm。

## 2. 机制结果总览

| # | 机制 | 运行 | 关键定量 |
|---|------|------|----------|
| M1 | 拉伸驱动滑移 | A3b, D2 | 2% 应变滑移 ~4a₀；D2 轨迹全程记录 |
| M2 | 偶极子湮灭 | B2 | 9.68% 应变 ±b 对湮灭（2→0） |
| M3 | **位错增殖** | D2 旗舰 | **10.50% 应变 4→8**，固定应变续弛豫 2000 步验证持久；阈值 σ/E≈7% |
| M4 | GB 塑性 + 率敏感 | B3/B5 | 多晶模量为单晶 39%；流应力 0.0057/0.0036/0.0026（率 4×↑/基准/4×↓）→ **粘塑性率指数 m=0.28**（对应金属扩散/GB 主导蠕变-超塑性区间）；快应变率下位错保有量更高（148/122/102 @8.5%） |
| M5 | 噪声辅助均匀形核 | B4 | noise 0.02 不形核（到 10.5%）；**noise 0.06 在 5.9% 形核**，峰值 18 核心 |
| M6 | **孔洞边缘异质发射** | B4 pore_v2 | 持久孔洞（depth 0.5、8a₀）边收缩边发射：2% 时 12 核心、6.2% 时 21 核心晕环、10.5% 孔耗尽留 18 位错簇——"位错工厂" |
| M7 | 孔洞 Gibbs-Thomson 溶解 | B4 pore_v1 | 小/浅孔洞（depth 0.2、4a₀）~6% 应变前溶解愈合 |

**3D（BCC）**：结晶/弹性拉伸门禁通过（C1/C2）；128³ 多晶（3290 原子）模量为单晶 75%、应力-应变呈早期塑性弯折（C3）。

**ROI 密度匹配（多尺度对接核心）**：DAMASK 热点 ROI ρ≈1.5-2.05×10¹⁵ m⁻² → 密度匹配 PFC 盒仅需 40 nm（1400² 网格），本地可跑。D2 旗舰即按 ROI 单元峰值密度配置（4 核心/42.5 nm = 2.2×10¹⁵），其增殖事件已按 CU 框架 DDD event proxy 格式输出（nucleation_count、threshold_strain）。

## 4. 已知局限与待办

1. 应力为加载路径功共轭量（PFC 无量纲单位），物理标定需弹性常数换算（C11-C12 等）——可与 DAMASK ROI 的 σ_vm(t)（峰值 ~1380 MPa）做比例对齐
2. 孔洞几何自动检测阈值不可靠（视觉确认正确）→ 改进 pore_geometry
3. 3D 位错线检测未实现（需 3D Burgers 回路或导出 OVITO/DXA）
4. PFC 扩散动力学时间尺度 ≠ MD 惯性时间尺度：应变率只能做相对比较
5. b4_noise_v2 中 cores=1 的奇数计数 = 偶极子部分检出（配对距离阈值）
6. HPC 不可达贯穿全程（7 轮重试）；大规模扫描（r×率×密度矩阵、3D 256³）待 HPC 可用

## 5. 与现有项目的衔接

- **CU 框架**：本工作 = Layer 4b（PFC 支线）首次实装；Interface B 读取器 + 密度种子映射 + 事件指标格式对齐已完成（D1/D2）
- **单晶铜拉伸模拟**：孔洞拉伸设置呼应其弧形缺口几何；σ(ε) 指标族与其 GROD/滑移分析互补
- **Topology research**：5|7 检测方法同源；模量/峰值强度/韧性指标对齐其 analyze_tensile_scan.py
- **BJK_hBN_Multiscale**：XPFC 谱方法数值风格的金属域移植
