# 如何让 PFC 森林硬化：分析与路径（2026-06-14）

5 代理工作流（文献+sessility物理+诊断模拟+XPFC可行性+综合）分析"怎么解决 PFC 不硬化"。

## 决定性结论：根因是动力学，不是晶体结构

**文献铁证**：没有任何 PFC 变体（经典/XPFC/两模/振幅）发表过森林或 Taylor 硬化。
- Berry-Provatas-Rottler-Sinclair PRB 86 (2012)：XPFC+两模+宽核三变体都做出 FCC 层错+Shockley 分位错——**无硬化**，LC 锁仅列为 future work。
- Berry et al. PRB 89 (2014)：做出 sessile LC stair-rod 锁，但只作预置障碍，可动位错 **Orowan 绕过**，无 flow-stress-vs-密度硬化曲线。
- 会硬化的"相场"是另一模型类（Koslowski/PFDD）：glide-on-prescribed-planes、**无攀移**、γ面+障碍内置。

**诊断模拟实测**：森林位错（60/120°系统）在 8% 剪切下净位移 4.01a₀（无驱动噪声底 2.44a₀）、
核数 6→4（被剪切驱动湮灭，无驱动对照恒为 6）——**森林可动+湮灭，非 sessile**。

**根因**：守恒 Model-B 只有一个扩散通道 ∇²μ 同时驱动滑移与攀移，无法分离，攀移永远开着。
→ 森林可动、反号湮灭、结点靠攀移解锁 → 加位错=加软模=软化。
**MPFC 没救回因为它只加速弹性通道，位错运动仍是同一扩散流。**
XPFC/两模改能量景观（给可锁产物）但保留这套动力学 → 仍会软化（裸跑会烧几个月重新软化）。

## 推荐路径（门控两阶段）
- **Stage 1（最便宜决定性测试，数天-2周，现有代码无需 XPFC）**：二元结点 climb-on/off。
  两条相交线形成结点，比较攀移抑制（高 beta+少扩散子步/增量）vs 攀移激活（多子步）下结点存活。
  判据：抑制攀移时 sessile 抗剪、激活时解锁 → 动力学诊断确认 → 进 Stage 2；
  若抑制攀移仍被绕过 → 晶体结构也卡 → XPFC/两模成共需，更难。
- **Stage 2（仅 Stage 1 确认后）**：抑制攀移动力学 + 两模 FCC（Wu，比 XPFC 便宜）供真分位错+LC 锁。
- **并行**：现在就发表已验证的负结果（首个明确的"守恒 PFC 不森林硬化"，2D+3D+稀疏+MPFC+256³）
  + PFC 能供给的高温蠕变本构量（迁移率形式、湮灭恢复、m(r)）。

## 工作量与风险
- Stage 1 go/no-go：数天-2周本地。Stage 2 若绿灯：周级到稳定两模 FCC，多月到可发表硬化曲线。
- 最深风险：原则性 glide-only 守恒动力学非平凡（投影扩散流到滑移方向可能破坏质量守恒/标架不变）；
  且可能"抑制攀移必要但不充分"（浅 Peierls 仍被绕过）→ 需动力学+晶体结构双修 = 多月。
- 诚实基率：无任何已发表 PFC 硬化 → 可能确认根本限制而非可修。缓解=便宜门控测试先分离动力学 vs 晶体结构。

## Stage-1 结点 climb 测试结果（2026-06-14，含诚实校正）
96³ 二元结点，攀移抑制(20子步) vs 激活(300子步)，同累计剪切 6%：
| | frac 演化 | 线 | τ_flow |
|--|---------|----|--------|
| 攀移抑制 | 0.342→0.287（缺陷保留） | 5→5 | 0.00106 |
| 攀移激活 | 0.342→0.000（全湮灭） | 5→0 | 0.00146 |

**确认**：攀移驱动湮灭/解锁（抑制→缺陷存活，激活→湮灭成完美晶体）。动力学诊断成立。
**自动判定"更硬"是假象**（已校正）：激活案例 τ_flow 更高是因湮灭成完美晶体（无位错=弹性高应力），
非森林硬化；保留缺陷的抑制案例反而更软（0.00106）——保留的森林仍是可滑移软模，未锁成抗剪结点。
**诚实结论**：抑制攀移让森林存活但**单独不足以硬化**；且"子步数"代理把应变率与攀移混淆，
不能干净隔离 climb-frozen 极限。干净判定+任何硬化希望需**真正实现 glide-only 守恒动力学**
（深层技术风险）+ 很可能两模 FCC = **确认多月工程**。
**便宜门控测试的价值**：确认湮灭机制，并阻止了"只抑制攀移就够"的天真尝试——动力学+晶体结构需双修。

## Stage-2a 抑制攀移迁移率：定量结果（2026-06-14）
实现高通迁移率核 G(k)=k⁴/(k²+k_c²)（压制长程攀移、保留晶格滑移），重跑 2D 森林硬化：
| kc | α | 
|----|---|
| 0（基线） | −0.223 |
| 0.5 | −0.179 |
| 1.0 | −0.150 |
**climb 抑制单调减少软化（证实攀移确实贡献软化），但不足以翻正到硬化。**
且 kc 有根本上限：k⁴/(k²+kc²) 在 kc>1 把晶格尺度(k≈1)滑移也压制（kc=1 已砍半 k=1 迁移率）
——攀移与滑移在 k≈1 尺度重叠，简单 k 滤波分不开（验证了"几何分解才是正路"的预判）。
**结论**：动力学旋钮必要且有用但单独不够 → 晶体结构修（两模 FCC sessile 锁）是必需第二修。
真正的 glide-only 需实空间逐位错几何分解（深层工程），k 滤波只是廉价上界探针。

## DUAL-FIX 决定性结果：仍软化（2026-06-14）
两模方晶（mu=0.197，相交滑移系）+ 抑制攀移 glide_kc，流应力 vs 种子密度：
| kc | d(tau)/d(√N) |
|----|------|
| 0.0 | −0.00136（软化） |
| 1.0 | −0.00131（软化） |
**两个修一起也没硬化。** 流应力随密度单调下降。

## 程序总判定：所有可及变体/修法均软化
| 变体/修法 | 结果 |
|---|---|
| 2D 三角 | 软化 α=−0.22 |
| 3D BCC 稀疏 256³ | 软化 |
| climb 抑制(k滤波) | 减轻但仍软化 (−0.22→−0.15) |
| 两模方晶 | 软化 |
| 两模+climb 双修 | 软化 |
**强证据：守恒 PFC 类不森林硬化**（与文献无任何 PFC 硬化过一致）。
**唯一未测组合**：3D XPFC-FCC（真分位错+LC stair-rod 锁）+ 真正实空间几何 glide-only 动力学
——最重的多月构建，且先验低（所有可及变体都软化）。两个所测的修都是真实尝试（非纸面），均失败。
**诚实结论**：这是有深层机制支撑的可发表负结果；硬化闭环在 PFC 框架内大概率不可达。
