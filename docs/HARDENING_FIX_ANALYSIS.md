# 如何让 PFC 森林硬化：分析与路径（2026-06-14）

5 代理工作流（文献+sessility物理+诊断模拟+XPFC可行性+综合）分析"怎么解决 PFC 不硬化"。

## 决定性结论：根因是动力学，不是晶体结构

**文献铁证**：没有任何 PFC 变体（经典/XPFC/两模/振幅）发表过森林或 Taylor 硬化。
- Berry-Provatas-Rottler-Sinclair PRB 86 (2012)：XPFC+两模+宽核三变体都做出 FCC 层错+Shockley 分位错——**无硬化**，LC 锁仅列为 future work。
- Berry et al. PRB 89 (2014)：做出 sessile LC stair-rod 锁，但只作预置障碍，可动位错 **Orowan 绕过**，无 flow-stress-vs-密度硬化曲线。
- 会硬化的"相场"是另一模型类（Koslowski/PFDD）：glide-on-prescribed-planes、**无攀移**、γ面+障碍内置。

**诊断模拟实测**：森林位错（60/120°系统）在 8% 剪切下净位移 4.01a₀（无驱动噪声底 2.44a₀）、
核数 6→4（被剪切驱动湮灭，无驱动对照恒为 6）——**森林可动+湮灭，非 sessile**。

**根因**：守恒 Model-B 只有一个扩散通道 ∇²μ 同时驱动滑移与攀移，无法分离，攀移永远开着。
→ 森林可动、反号湮灭、结点靠攀移解锁 → 加位错=加软模=软化。
**MPFC 没救回因为它只加速弹性通道，位错运动仍是同一扩散流。**
XPFC/两模改能量景观（给可锁产物）但保留这套动力学 → 仍会软化（裸跑会烧几个月重新软化）。

## 推荐路径（门控两阶段）
- **Stage 1（最便宜决定性测试，数天-2周，现有代码无需 XPFC）**：二元结点 climb-on/off。
  两条相交线形成结点，比较攀移抑制（高 beta+少扩散子步/增量）vs 攀移激活（多子步）下结点存活。
  判据：抑制攀移时 sessile 抗剪、激活时解锁 → 动力学诊断确认 → 进 Stage 2；
  若抑制攀移仍被绕过 → 晶体结构也卡 → XPFC/两模成共需，更难。
- **Stage 2（仅 Stage 1 确认后）**：抑制攀移动力学 + 两模 FCC（Wu，比 XPFC 便宜）供真分位错+LC 锁。
- **并行**：现在就发表已验证的负结果（首个明确的"守恒 PFC 不森林硬化"，2D+3D+稀疏+MPFC+256³）
  + PFC 能供给的高温蠕变本构量（迁移率形式、湮灭恢复、m(r)）。

## 工作量与风险
- Stage 1 go/no-go：数天-2周本地。Stage 2 若绿灯：周级到稳定两模 FCC，多月到可发表硬化曲线。
- 最深风险：原则性 glide-only 守恒动力学非平凡（投影扩散流到滑移方向可能破坏质量守恒/标架不变）；
  且可能"抑制攀移必要但不充分"（浅 Peierls 仍被绕过）→ 需动力学+晶体结构双修 = 多月。
- 诚实基率：无任何已发表 PFC 硬化 → 可能确认根本限制而非可修。缓解=便宜门控测试先分离动力学 vs 晶体结构。
