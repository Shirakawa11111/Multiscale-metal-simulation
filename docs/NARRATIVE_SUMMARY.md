密度 ~1.5e15 m⁻²）与主线对接。科学问题：拉伸/剪切下位错的形核-滑移-湮灭机制。

## 第 5 幕 — 硬化森林攻坚（用户：试着解决，别只陈述限制）
**判决性确立 PFC 不森林硬化**，且逐层定位根因：
| 尝试 | 结果 |
|---|---|
| 2D 三角森林 | 软化 α=−0.22 |
| 3D BCC 稀疏（256³ HPC，4.2h×6） | 软化 |
| 抑制攀移（高通迁移率核 k⁴/(k²+k_c²)） | 减轻但仍软化（−0.22→−0.15）；k 滤波因攀移/滑移尺度重叠有根本上限 |
| 两模方晶（相交滑移系，找到稳健参数区） | 软化 |
| 双修（两模+抑制攀移） | 软化 |
| **钉扎森林（理想 sessile 极限）** | **判决性：软化消除(slope→~0)但仍不硬化** |
**机制分解**（钉扎诊断）：软化 = 森林可动/湮灭（**动力学**，可除）；不硬化 = 可动位错
**Orowan 绕过** sessile 障碍而非锁结点（**核结构**，根本）。完全复现 Berry 2014（XPFC 里真
LC 锁仍被绕过）→ 连最重的 3D XPFC-FCC 全构建先验也极低（终极阻碍在那里照样存在）。
文献铁证：**无任何 PFC 变体发表过森林硬化。** 每个修都是真实实现的尝试，终极阻碍被干净隔离。

## 第 6 幕 — 转回多尺度主线：PFC 作为高温蠕变桥梁
把 PFC 定位到它**物理上正确**的角色。非循环蠕变耦合（每输入独立来源）：
PFC 迁移率 v=M·b·τ → 扩散控制蠕变流动律 ε̇=ρbv，时间标定给温度依赖，ρ 来自 DAMASK ROI。
**结果**：应力指数 **n=1.00**（粘性/扩散蠕变）、表观激活能 **2.00 eV**（精确回收输入自扩散 Q）。

## 统一结论
**"PFC 不能森林硬化"与"PFC 擅长高温蠕变"是同一物理的两面**：守恒扩散动力学 = 高温/
蠕变区，那里森林硬化本就弱、扩散/攀移/晶界机制主导。完整诚实叙事 = 一篇论文骨架：
1. PFC 机制库（2D/3D，验证）
2. **负结果**：PFC 不森林硬化（判决性 + 钉扎机制分解 + 与文献一致）
3. **正结果**：PFC 是有效高温蠕变介观桥梁（迁移率 M=1.54、湮灭恢复、率敏感 m(r)；
   时间标定置扩散区；非循环 PFC→CP 蠕变耦合 n=1/Q=2eV）

## 方法学元教训（这次循环本身的收获）
- **对抗性验证是必需品**：抓出了 4 个致命缺陷 + 多次过度解读（"晶界滑移"、"增殖闭环"、
  自动判定的"更硬"假象）。单一指标趋势会骗人。
- **诚实负结果有价值**：试遍所有可及修法、用钉扎诊断隔离终极阻碍，比"放弃"或"硬吹"都强。
- **便宜门控测试**省下数月（避免盲目 3D XPFC-FCC）。
- **限制与贡献常自洽**：理解为什么 PFC 不能硬化，恰好揭示它能做什么。

## 局限与未竟
- 蠕变耦合绝对速率示意性偏高（管道扩散因子）；可辩护输出是 n=1 形式 + Q 回收（区制判定）。
- 真正的 glide-only 守恒动力学（实空间逐位错投影）未实现（钉扎是其理想极限的诊断代理）。
- Cu 是 FCC，本工作 2D 三角 + 单模 BCC；FCC 保真需 XPFC/两模（已知路径，本工作只用 b 尺度映射）。
- y_ann 需用力平衡/收敛判据重测才可定量引用。
