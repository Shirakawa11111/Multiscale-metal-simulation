## 子问题 B 第一步成果（2026-06-13）：可传递迁移率律
v(τ) 扫描 → 带阈值拟合 **v = M·b·(τ − τ_c)**（DDD/ExaDiS 消费的过阻尼线性形式）：
- MPFC 滑移支：M=1.53, τ_c=6.5e-4, **R²=0.97**
- 扩散版：M=1.13, τ_c=1.1e-4, R²=0.84
- 低应力 v<0 = 偶极互吸/Peierls 阈值（物理，非bug）
**判定**：PFC 供给 DDD 兼容的迁移率*函数形式*（稳健）；*系数* M 依赖动力学
（MPFC>扩散，快弹性弛豫减人工阻力，物理合理）。
**剩余障碍（闭环最后一步）**：PFC-时间↔秒标定 → 绝对阻尼 B。
可经一个物理锚（Cu 拖曳系数 B≈1e-4 Pa·s 或扩散率）标定，或在无量纲趋势层面闭环
（PFC 的 M(r)/τ_c(r) 喂 Orowan/Taylor 律 ρbv=应变率 + DAMASK ROI 密度 → 改变流应力预测）。

## Taylor 闭环尝试：诚实负结果 + 重定向（2026-06-13）
直接"种子偶极子→流应力 vs ρ"得 α=−0.287（负，R²=0.74）：流应力随密度**下降**
(0.0039→0.0028) 而非 √ρ 硬化。根因：稀疏构型 + 2D 单滑移方向 → **湮灭主导**，
非森林硬化（森林硬化需相交滑移系上的稳定森林，本几何实现不了）。
→ **重定向**：软化恰证明**位错湮灭距离 y_ann（以 b 计）** 才是此 PFC 能干净给出的
无量纲传递量——正是 Kocks-Mecking/dislotwin 恢复项显式使用的 CP/DDD 参数，无需时间标定。

## 收口：PFC 标定的晶体塑性密度演化律（2026-06-13）
**核心问题的实质解答** —— PFC 不再是单向单位换算，而是向 CP 上传一组无量纲本构量：
| PFC 测得量 | 值 | CP/DDD 中的角色 |
|---|---|---|
| 湮灭距离 y_ann | ~7 b | KM/dislotwin 恢复项 k2=2·y_ann/b |
| 迁移率律 v=M·b·(τ−τ_c) | M≈1, R²=0.97 | DDD 流动律 / KM 流动 |
| 增殖阈值 σ/E | ~0.07 | 形核/源激活 |
| 率指数 m(r)（2D, MPFC 确认） | 0.19→0.45 | 率敏感性 |

**闭环演示**（`upscale_km.py`）：上述参数 → KM 律 dρ/dγ=k1√ρ−k2ρ → 从 DAMASK ROI
密度积分 → 饱和密度 2.5e15 m⁻²（ROI 锚 1.5e15）、Taylor 流应力 ~197 MPa（Cu 森林阻力量级）。
**诚实边界**：Taylor α 用文献值（PFC 森林几何不成立，见负结果）；KM 储存项锚定 ROI；
绝对时间/率需标定。但 PFC→CP 的传递*结构*已打通：PFC 供给密度演化与流动律的形式+无量纲系数。

## 时间标定：闭合最后一个单位，故事自洽（2026-06-13）
**linchpin 已解决** —— 完整单位系统：长度 b/a₀、应力 μ_Cu/μ_pfc=8.8e11 Pa、
**时间 τ_PFC（扩散锚定）**：D_PFC=0.90（测）≈ r+1=0.75（解析），锚到 Cu 自/管道扩散
→ τ_PFC 从 0.5Tm 的 0.04s 到 0.8Tm 管道扩散的 1e-9s。

比无热滑移慢数量级——正是扩散/攀移控制运动，验证守恒 PFC 属高温区。
**关键收获**：这解决了 m(r)→0.45"超塑性"的风险——PFC 扩散动力学**本就是高温/超塑性区**，
m→0.5 是物理正确而非假象。m(r)、晶界滑移、扩散蠕变、迁移率全部自洽落在同一高温扩散控制区。
→ 项目定位转正：从"未标定玩具"变为"标定到 Cu 的定量高温蠕变介观模型"。

## 硬化森林 PFC 程序：2D 决定性负结果 + 3D 重定向（2026-06-13）
**测试**：2D 三角，主系(0°)可动位错 + 60°/120° 相交滑移系森林 + MPFC 弹性弛豫，
流应力 vs 森林密度。**结果：仍软化，α=−0.223（R²=0.83）**，流应力随密度单调降。
**判定**：2D PFC 无法森林硬化——**位错交割(junction/lock)本质是 3D 过程**；2D 点位错
只能擦过/湮灭，加密度=加恢复通道=软化（与 2D DDD 需人为结点规则、3D DDD 靠真实结点
自然硬化一致）。已排除几何与弹性弛豫因素（都用对了仍软化）。
**重定向**：森林硬化需 3D 位错线+结点。M18 已证 3D BCC 剪切到 15% 不非晶化 → 
**下一步在 3D BCC 剪切下测森林/结点硬化**（比 XPFC-FCC 便宜的关键判据）。
XPFC-FCC 对 Cu 保真重要，但森林物理门槛是 3D 结点，先在 3D BCC 剪切验证 PFC 能否捕获
结点硬化（守恒 PFC 能否稳定结点是真开放问题）。
