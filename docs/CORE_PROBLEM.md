## 子问题 B 第一步成果（2026-06-13）：可传递迁移率律
v(τ) 扫描 → 带阈值拟合 **v = M·b·(τ − τ_c)**（DDD/ExaDiS 消费的过阻尼线性形式）：
- MPFC 滑移支：M=1.53, τ_c=6.5e-4, **R²=0.97**
- 扩散版：M=1.13, τ_c=1.1e-4, R²=0.84
- 低应力 v<0 = 偶极互吸/Peierls 阈值（物理，非bug）
**判定**：PFC 供给 DDD 兼容的迁移率*函数形式*（稳健）；*系数* M 依赖动力学
（MPFC>扩散，快弹性弛豫减人工阻力，物理合理）。
**剩余障碍（闭环最后一步）**：PFC-时间↔秒标定 → 绝对阻尼 B。
可经一个物理锚（Cu 拖曳系数 B≈1e-4 Pa·s 或扩散率）标定，或在无量纲趋势层面闭环
（PFC 的 M(r)/τ_c(r) 喂 Orowan/Taylor 律 ρbv=应变率 + DAMASK ROI 密度 → 改变流应力预测）。

## Taylor 闭环尝试：诚实负结果 + 重定向（2026-06-13）
直接"种子偶极子→流应力 vs ρ"得 α=−0.287（负，R²=0.74）：流应力随密度**下降**
(0.0039→0.0028) 而非 √ρ 硬化。根因：稀疏构型 + 2D 单滑移方向 → **湮灭主导**，
非森林硬化（森林硬化需相交滑移系上的稳定森林，本几何实现不了）。
→ **重定向**：软化恰证明**位错湮灭距离 y_ann（以 b 计）** 才是此 PFC 能干净给出的
无量纲传递量——正是 Kocks-Mecking/dislotwin 恢复项显式使用的 CP/DDD 参数，无需时间标定。
