# 突破纲领：从第一性相互作用矩阵到实验网络的定量预测

目标：把项目从"范围界定 + 方法学警示"推进到一个**定量、有第一性物理锚定、连接真实实验**的结果，系统补全所有弱点。诚实定位：争取最强真实结果，是否够"突破"由结果自证，不预先贴标签。

## 要补的弱点 → 修复 → 交付
1. **α 非真实森林系数**（钉扎随机子集，未验证穿过滑移面）
   → Madec 式受控测量：单滑移系森林 + 单探针，逐结点类型测 a_ij
   → 完整 6 参数 FCC 矩阵；由它加权**复现体材料 α=0.3–0.5**
2. **STEM 耦合只是几何 Burgers / 合成 z**
   → 物理一致性反演：在几何容许的 Burgers 空间筛选；给稳健预测区间
3. **无新意（复现教科书）**
   → 矩阵 → 实验网络的**预测闭环**是新的；补 ρ_ss(ε̇)、开交滑移
4. **多尺度链是蓝图、MD 空缺**
   → MD 实测一个 DDD 参数（迁移率 B 或结点强度）喂进 DDD
